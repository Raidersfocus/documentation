Les Images à zone sensible

les images a zone sensible permet de définir une formes géométrique qui définissent des zones dans une image en utilisant les coordonnées X et Y

  • <img src="exemple.png" usemap="#exemple/nom de votre map">
    permet d'appliquer la map sur une image
  • <map name="exemple">...</map>
    permet de créer la map (ceci est obligatoir et doit être mis avant se qui va suivre !)
  • <map name="exemple">...<area type="rect" coords="x1, y1, x2, y2" href="reference">...</map>
    permet de faire une zone de forme rectangulaire à partir de 2 coordonnées
  • <map name="exemple">...<area type="circle" coords="x1, y1, r" href="reference">...</map>
    permet de faire une zone de forme circulaire à partir de sa position et de son rayon
  • <map name="exemple">...<area type="poly" coords="x1, y1, x2, y2, x3, y3, xn, yn..." href="reference">...</map>
    permet de faire une zone en indiquant tout les sommets de la forme voulu

Les Balises sémantique

Les Balises sémantique permettre de reconnaitre les partie de notre code HTML et permette aussi de rejouté du style avec le language CSS

Voici une petite liste :

  • <section>...</section>
    Section de page, la pluspart des balises sémantiques sont prévues pour être utilisées dans la balise section
  • <article>...</article>
    A utiliser pour définir un article, un commentaire, ...
  • <aside>...</aside>
    Texte rapporté à une section mais avec un conteneur propre (cas de l'encart attenant à un article de journal)
  • <header>...</header>
    Entête de la page au niveau du contenu (à ne pas confondre avec <head>)
  • <hgroup>...</hgroup>
    groupe de titres (h1...h6) par exemple pour le sommaire
  • <footer>...</footer>
    Pieds de page
  • <nav>...</nav>
    A utiliser pour le menu
  • <figure>...</figure>
    A utiliser pour une image accompagnée de sa légende
  • <figcaption>...</figcaption>
    Définit la légende à associer à une balise

Intégré du code CSS directement dans le code HTML

Pour Intégré du code CSS sans passer par un fichier externe, il faut entrer la balise <style>...</style> dans la balise <head>...</heade>


<!doctype html>
<html lang="fr">
	<head>
		<meta charset="UTF-8">
		<title>titre de la page</title>
		<style type="text/css">
			<!--code CSS-->
		</style>
	</head>
	<body>
		<!--code HTML-->
	</body>
</html>

Utiliser du code CSS dans un fichier externe

Pour Intégré du code CSS en passant par un fichier externe, il faut entrer la balise <link rel="stylesheet" type="text/css" href="le chemin de votre fichier"> dans la balise <head>...</heade>


<!doctype html>
<html lang="fr">
	<head>
		<meta charset="UTF-8">
		<title>titre de la page</title>
		<link rel="stylesheet" type="text/css" href="le chemin de votre fichier">
	</head>
	<body>
		<!--code HTML-->
	</body>
</html>

Syntax du CSS
  • Le sélecteur :
    Il permet, comme son nom l'indique, de selectionner une balise avec son nom comme, <h1>, <a>, <ul>, <pre>, <p>, etc...
    On peut aussi infecter plusieur balises en même temp, soit en utilisant la selection multiple, le sélécteur de classes ou le sélécteur d'id
    block de déclaration :
    ce sont les { }
    Déclaration :
    Les déclaration sont les arguments qui affectera les balises : color: red; (mettra le texte en rouge)
    Propriété :
    Sera la façon dont vous affecterez les balises : color (permet de changer la couleur d'un text)
    La valeur :
    Permet de régler la façon dont vous affecterez une balise
  • Exemple :
    
balise {
	color: red;
}
    Change la couleur de texte en rouge en affectant une seule balise
    
balise1, balise1 {
	border: 1px solid blue;
}
    rajoute une bordure en affectant plusieurs balises avec la selection multiple
    
HTML :
	<balise class="exemple">...</balise>

CSS :
	.exemple {
		background: green;
	}
    Change la couleur de font en vert en affectant toute les balises qui ont la class "exemple" (class="exemple")
    
HTML :
	<balise id="exemple">...</balise>
	
CSS :
	#exemple {
		background: violet;
	}
    Change la couleur de font en violet en affectant seulement la balises qui a l'id exemple (id="exemple")

Rendre votre site "responsive"

Rendre votre site "responsive" veux dire que le site pourra détécter la taille de votre écran pour pouvoir adapter votre site a tout les appareils comme smartphone, tablette, etc...

  1. <meta name="viewport" content="width=device-width, initial-scale=1.0, shrink-to-fit=no">
    Pour récupérer la résolution de l'écran de l'utilisateur
  2. <link rel="stylesheet" media="screen and (max-width: (la résolution limite)px)" href="mobile.css">
    Puis préparer et lier votre feuille de style
    le tout est mis dans <head>...</head>

Quand la résolution dépassera (la limite de résolution) le site prendra la feuille de style mobile.css

Spécificité

La spécificité permet au navigateur de prioriser une mise en style avec une règle

La priorité sera representer comme cela : 0, 0, 0, 0

Le plus grand chiffre de gauche a droite sera pris comme prioritaire et afficher, donc : 0, 0, 1, 0 > 0, 0, 0, 255

Pour chaque valeur de l'attribut id donné dans le sélecteur, ajouter 0,1,0,0

Pour chaque valeur de l'attribut class de sélection d'attribut ou de pseudo-classe donné dans le sélecteur, ajouter 0,0,1,0

Pour chaque élément et pseudo élément donné dans le sélecteur, ajouter 0,0,0,1

  • Exemple :
    
h1 {color:red;} // 0, 0, 0, 1
body h1 {color:green;} // 0, 0, 0, 2
    Le deuxième sélécteur sera utiliser pour afficher le style, parce que sa spécificité est plus grand
    
h2.grappe {color:red;} // 0, 0, 1, 1
h2 {color: sliver;} // 0, 0, 0, 1
    Le premier sélécteur sera utiliser pour afficher le style, il y a un chiffre plus a gauche que la deuxième, donc il est prioriser
    
body table tr td ul {color: maroon;} // 0, 0, 0, 5 
li#reponse {color:navy;} // 0, 1, 0, 1
    Le deuxième sélécteur sera utiliser pour afficher le style, il y a un chiffre plus a gauche que la deuxième, donc il est prioriser
