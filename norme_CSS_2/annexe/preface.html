Corrections normatives

La traduction reprend toutes les corrections répertoriées dans la page des errata qui accompagne chaque document. Le cas échéant, ces corrections apparaissent de deux manières différentes :

  • Soit directement dans le texte : voici un ajout, voilà une suppression ;
  • Soit sous forme d'un renvoi vers l'entrée correspondante dans la page des errata, par exemple : « errata E01 ».

IMPORTANT :

Assurez-vous de l'actualité des corrections dans la traduction.
